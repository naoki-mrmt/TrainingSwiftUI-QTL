## 環境構築
### 事前準備
- Bundlerをインストール
    - ```gem install bundler```
    - [rbenv](https://github.com/rbenv/rbenv) が入ってると楽
- CocoaPodsのインストール
    - ```bundle install```
 
### xcworkspaceの生成
- ```make```
    - 中身 
        - ```bundle exec pod install```
        - ```open ./Task4.xcworkspace```
